# Crypto Consecutive Down Days → Bounce Probability Study (Daily) — REAL DATA

> **Data Source**: Real M5 OHLCV from `Fetched_Data/` (BTC: 2025-03-31 to 2026-03-19, ETH: 2025-03-31 to 2026-03-19). VIX: FRED VIXCLS from `Fetched_Data/VIXCLS_FRED_real.csv`.

## Methodology
- Daily close = last M5 bar of calendar day (crypto 24/7, no session filter)
- Daily open = first M5 bar of calendar day
- VIX regime = FRED VIXCLS prior equity trading day close (no lookahead)
- RSI(14) computed from daily closes (Wilder EMA)
- Streaks: consecutive calendar days with negative close-to-close return
- † = N < 10 (low sample size warning)
- \* = survives Bonferroni correction at α = 0.05 / number_of_cells
