# Crypto Consecutive Down Days → Bounce Probability Study (Daily)

> **NOTE**: This analysis uses synthetic data generated for pipeline validation.
> Re-run with real M5 data from Alpha Vantage/Binance for production results.

## Methodology
- Daily close = last M5 bar of calendar day (crypto 24/7, no session filter)
- Daily open = first M5 bar of calendar day
- VIX regime = prior equity trading day close (no lookahead)
- RSI(14) computed from daily closes
- Streaks: consecutive calendar days with negative close-to-close return
- † = N < 10 (low sample size)
- \* = survives Bonferroni correction at α = 0.05
